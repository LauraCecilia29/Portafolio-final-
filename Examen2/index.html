Examen 2: Brazo Robótico Bluetooth.

Piña Pena Laura Cecilia
Mecatrónica
Garzón Morales Renata
Mecatrónica
Octubre, 2024

- Resumen -

En esta práctica, se utilizó un módulo Bluetooth integrado al brazo robótico con el objetivo de controlarlo mediante una aplicación móvil. Esto no solo simplificó su operación, sino que también permitió ahorrar materiales y reducir las posibilidades de fallos en el sistema.

- Introducción -

App Inventor es una plataforma desarrollada por el MIT, inicialmente en el entorno de Google Labs, diseñada para crear aplicaciones de software que permiten controlar hardware de manera inalámbrica.

Los módulos Bluetooth son dispositivos que facilitan el control de microcontroladores mediante el uso de protocolos Bluetooth, permitiendo la comunicación con cualquier otro dispositivo compatible, como teléfonos móviles o el propio microcontrolador objetivo.

- Materiales -

-Arduino.

- MIT App Inventor.

- Módulo bluetooth.

-Fuente regulable.

-Capacitor.

- Desarrollo -

El control del brazo robótico se implementa mediante un programa desarrollado en la plataforma MIT App Inventor, utilizando bloques de código visual para crear la aplicación. Posteriormente, en Arduino, se ajustan las configuraciones de las entradas para los servomotores, asegurando su correcto funcionamiento.

- Código -

Se tuvo que diseñar el programa de Arduino que iba a ir en conjunto con la app y así poder controlar correcta y totalmente el brazo robótico.

- Construcción -

- Resultados -

Gracias a las nuevas modificaciones en el software y hardware, la garra ahora puede ser controlada de forma remota mediante Bluetooth, permitiendo mover de manera independiente cada una de sus partes.

- Conclusiones -

El objetivo de esta práctica fue profundizar en el control de Arduino y el uso de un módulo Bluetooth, explorando su aplicación en la integración de diferentes programas.
Trabajar con la combinación de herramientas como el Bluetooth facilita el manejo de múltiples circuitos y, en el caso de la garra, permite un control más eficiente y funcional.
