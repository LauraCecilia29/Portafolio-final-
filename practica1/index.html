- Resumen -

Con esta práctica se busca que se entienda mejor como contruir un circuito.

- Introducción -

Durante está actividad se aprenderá como generar un circuito en parejas con ayuda de thinkercard, y componentes eléctricos.

- Materiales -

Una computadora

Protoboard

Leds

Cables

Pila

Resistencias

Potenciometro

- Desarrollo -

Cuando se realizó esta actividad se buscaba poder entender mejor como conectar un circuito en físico como en thinkercard. La idea es que al girar el potenciometro hacia un lado bajara la velocidad en la que parpadeaba el led, y si se giraba el potenciometro para el otro lado aumentar la velocidad en la que parpadeaba el led.

Construcción

Durante la construcción cosideramos que fue sencilla y no tuvimos ningún problema.

Realizamos todo lo que se fue explicando durnate la clase y conctamos los componentes necesarios para la creación del led que parpadea.

El circuito funcionó tal y como se esperaba.
